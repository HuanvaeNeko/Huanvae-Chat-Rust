# Web 测试界面 - 使用说明

## 📝 概述

这是一个用于测试 HuanVae Chat API 的网页测试工具，支持用户认证、好友管理和好友消息功能。

## 🚀 使用方法

### 1. 启动后端服务

```bash
cd /home/huanwei/Huanvae-Chat-Rust
cargo run
```

### 2. 打开测试页面

在浏览器中打开:
```
file:///home/huanwei/Huanvae-Chat-Rust/test-web/index.html
```

或者使用简单的 HTTP 服务器（推荐）:
```bash
cd /home/huanwei/Huanvae-Chat-Rust/test-web
python3 -m http.server 3000
# 然后访问 http://localhost:3000
```

## 📋 功能模块

### 1. 后端设置
- 配置 Auth API 地址（默认: `http://localhost:8080/api/auth`）
- 配置 Friends API 地址（默认: `http://localhost:8080/api/friends`）
- 配置 Messages API 地址（默认: `http://localhost:8080/api/messages`）

### 2. 用户认证
- **注册**: 创建新用户账号
- **登录**: 获取 Access Token 和 Refresh Token
- **查看个人信息**: 解析 JWT Token 查看用户信息
- **刷新 Token**: 使用 Refresh Token 获取新的 Access Token

### 3. 设备管理
- **查看设备列表**: 查看当前用户所有登录设备
- **删除设备**: 远程登出指定设备

### 4. 好友功能
- **发送好友请求**: 向其他用户发送好友申请
- **查看待处理请求**: 查看收到的好友申请
- **同意/拒绝申请**: 处理好友请求
- **查看已发送请求**: 查看自己发出的好友申请
- **查看好友列表**: 查看所有好友
- **删除好友**: 移除好友关系

### 5. 好友消息功能 ✨ (新增)

#### 发送消息
- **接收者ID**: 输入好友的用户ID
- **消息类型**: 
  - `text` - 文本消息
  - `image` - 图片消息
  - `video` - 视频消息
  - `file` - 文件消息
- **消息内容**: 输入要发送的内容
- **文件URL** (可选): 媒体消息时填写文件地址
- **文件大小** (可选): 文件大小（字节）

#### 获取消息列表
- **好友ID**: 查看与该好友的聊天记录
- **起始消息UUID** (可选): 用于分页，从指定消息之前查询
- **查询数量**: 默认50条，最大500条
- **消息显示**: 
  - 绿色背景 = 发送的消息
  - 蓝色背景 = 接收的消息
  - 显示发送时间、消息UUID
  - 提供删除和撤回按钮

#### 删除消息（软删除）
- **消息UUID**: 输入要删除的消息ID
- **效果**: 只对当前用户生效，对方仍可看到

#### 撤回消息
- **消息UUID**: 输入要撤回的消息ID
- **限制**: 
  - ⏰ 只能撤回2分钟内的消息
  - 🔒 只有发送者可以撤回
  - 🗑️ 撤回后双方都看不到

## 🎯 典型测试流程

### 测试好友消息功能

```
步骤1: 准备两个测试账号
  1. 注册用户A (user_id: alice, password: Password123)
  2. 注册用户B (user_id: bob, password: Password123)
  
步骤2: 登录用户A
  1. 输入 alice / Password123
  2. 点击"登录"
  3. Token 会自动保存到浏览器

步骤3: 用户A添加用户B为好友
  1. 在"好友功能"区域
  2. 目标用户ID输入: bob
  3. 点击"发起好友申请"
  
步骤4: 切换到用户B（可以开另一个浏览器窗口）
  1. 登录 bob / Password123
  2. 点击"获取待通过申请"
  3. 点击"同意"按钮
  
步骤5: 用户A发送消息给用户B
  1. 在"好友消息功能"区域
  2. 接收者ID输入: bob
  3. 消息内容输入: Hi Bob!
  4. 点击"发送消息"
  
步骤6: 用户B查看消息
  1. 好友ID输入: alice
  2. 点击"获取消息列表"
  3. 可以看到来自alice的消息
  
步骤7: 用户B回复
  1. 接收者ID输入: alice
  2. 消息内容输入: Hi Alice!
  3. 点击"发送消息"
  
步骤8: 测试删除和撤回
  1. 在消息列表中点击"删除"按钮
  2. 或在2分钟内点击"撤回"按钮
  3. 刷新消息列表查看效果
```

## 💡 提示

### 消息功能注意事项
1. **好友关系**: 只能向好友发送消息，非好友会返回错误
2. **Token 有效期**: Access Token 15分钟过期，需要刷新
3. **消息查询**: 使用分页查询可避免一次加载过多消息
4. **撤回时限**: 消息发送超过2分钟后无法撤回
5. **自动刷新**: 发送、删除、撤回成功后会自动刷新消息列表

### 调试技巧
1. **查看请求**: 每个操作都会显示完整的请求和响应
2. **本地状态**: 底部显示当前Token状态
3. **消息UUID**: 每条消息都显示UUID，可用于删除/撤回
4. **加载更多**: 如果有更多消息，会显示"加载更多"按钮

### 常见问题

**Q: 为什么无法发送消息？**
- 确保已登录（有Access Token）
- 确认双方已经是好友关系
- 检查接收者ID是否正确

**Q: 为什么看不到消息？**
- 确认好友ID输入正确
- 检查是否有消息被软删除
- 确认Token未过期

**Q: 撤回消息失败？**
- 只有发送者可以撤回
- 消息必须在2分钟内
- 确认消息UUID正确
